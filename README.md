# El Goloso Web

Página web estática de El Goloso para publicar en Vercel.

## Archivos

- `index.html`: entrada principal del sitio para Vercel.
- `code.html`: archivo HTML original recibido.
- `DESIGN.md`: guía de diseño original.
- `screen.png`: captura de referencia original.

## Despliegue

El proyecto no necesita proceso de build. Vercel puede publicarlo como sitio estático desde la raíz del repositorio.
